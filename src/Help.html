BEGINNING OF YEAR SETUP

Copy the Templates, Create the Folders, Set Up the Database

  1. Create a main folder in your Google Drive that will house all of the project folders and files that you will create in the steps below.
  2. Make a copy of the Excel Student Profiles Google Spreadsheet template at: https://docs.google.com/spreadsheets/d/1Xn0kIGAcfEItonYtEe1lp44IlnEnwZRkbatNmwMp7Zw/copy and place it inside your newly created folder. This is the spreadsheet that will serve as your database.
  3. Make a copy of the Excel Profiles Google Doc template at: https://docs.google.com/document/d/1EMIjCPSyxfVeyFQncplevMR66RRb_pNv0yqnJ27EJjg/copy and place it inside the newly created folder. This is the file that AutoCrat will use to create the profile docs that the teachers will be able to see.
  4. From inside the main folder, create another folder where the picture images (BMP files) will be saved. You will upload the student .bmp picture files into this folder. Copy this folder's id because you will use it in the next step. If you need help finding the folder ID, go to the REFERENCES section below and you'll see some information on that.
  5. Return to the Excel Student Profiles Google Spreadsheet, go to the Setup sheet, and paste the folder's id into cell B3.
  6. Go back to the main folder and create another folder where the JPGs will be stored (the converted BMP files). Copy this folder's id for the next step.
  7. Return to the Excel Student Profiles Google Spreadsheet, go to the Setup sheet, and paste the JPG folder's id into cell B4.
  8. Go back to the main folder and create a third folder where the teachers will see the profile pages. Remember the name of this folder because you will need it when you set up AutoCrat below.
  9. Share the profile pages folder with the teachers and staff that need access to the profile pages. Click on the Share button in the top right, add their names, and give them "Viewer" permission.
  10. Return to the Excel Student Profiles Google Spreadsheet and update the Adv. column. Click on Data then Data validation. A side menu pops up to the right of the screen. Click on the range that looks something like, C2:C751. A list of teacher initials opens up. Adjust the list by deleting the ones that are not needed and adding new ones. Once this list is updated, look at the Apply to range field and adjust the formula in the field to Enrolled!C2:C so that it includes the dropdown in all of the cells in column C. Click the green Done button at the bottom right. Click the X to the right of Data validation rules to close the side menu.

AutoCrat Setup

Think of AutoCrat as a mail merge between Google Sheets and Google Docs. AutoCrat is a Chrome extension that should already be installed on your Google Sheet account. You can verify that it is installed by going to the Excel Student Profiles Google Spreadsheet then clicking on Extensions in the top menu. If you do not see AutoCrat listed, you will need to install it by following this link: https://workspace.google.com/marketplace/app/autocrat/539341275670 and clicking Install.

Once AutoCrat has been installed, complete the next steps to create the AutoCrat job. You only need to follow the steps below to create the job initially. Once the job is created you just need to run it throughout the year.

  1. From the Enrolled sheet, click on Extensions, AutoCrat, then Open. The AutoCrat dialog window opens.
  2. Click on NEW JOB at the bottom right.
  3. Give the job an easily recognizable name. Then click on the blue Next button at the bottom right.
  4. To choose the template, click on the blue From drive button then search for the template you downloaded in step 3 above. Once you find the template, double-click on it to select it then click on the blue next button.
  5. The next step is to map the source data to the template. In the Merge tab dropdown, select Enrolled. This is the name of the sheet that the data will come from. The next step involves matching the dropdowns on the right column to the tags from the template on the left column. For example, << Images >> is on the left column, so on the field immediately to the right you will click on the dropdown and select "Images". This "maps" the data, in other words, it tells AutoCrat what data to insert into the template. Repeat this step until all of the fields on the right side are mapped to a tag on the left column. Once done mapping, click on the blue Next button.
  6. In the next screen, you will tell AutoCrat how to name each file it creates. For example if I want to name the files with the student's name then their ID, I would click inside the File name field then click in the blue bar to the left of it. A menu listing the tags will expand. Click on one of those tags to copy it to the clipboard. In this example I would click on <>. The blue bar minimizes. Next, click inside the File name field and CTRL + V to paste the tag. Next, type a comma then add a space. Next, click on the blue bar again and click on the <>. Click on the File name field again and CTRL + V to paste the tag. Click on the blue Next button.
  7. Choose the folder you created in Step 8 above. This is where the profiles will be placed after they are created. Click the blue Next button.
  8. Skip the step to add a dynamic folder. Click on the blue Next button.
  9. Skip the merge condition step. Click on the blue Next button.
  10. Skip the share docs & send emails steps. Click on the blue Next button.
  11. In the add/remove job triggers make sure they are set to "No". Click on the blue Next button.
  12. Click on the red SAVE button.

THROUGHOUT THE YEAR

As the year goes on, you will need to update the Excel Student Profiles Google Spreadsheet then run two automations.

One automation converts student .bmp picture files into .jpg format. The conversion is necessary for Google Docs to use when making the profile docs. After the conversion is made, the script gets the hyperlink of the .jpg file and inserts it into column L of the Excel Student Profiles Google Spreadsheet.

The second automation is done using AutoCrat. AutoCrat merges the .jpg image and student information into the profile document.

Below are the steps for performing these two automations.

  1. As students enroll at Excel, update the data in columns A through K in the Enrolled sheet.
  2. Click on the Get Picture Hyperlinks menu then Import Picture Hyperlinks. This step runs a script in the background that converts the .bmp files into .jpgs and pastes the hyperlink into column L.
  3. Go to Extensions then AutoCrat then Open. Give it a few seconds to open Existing Jobs.
  4. Look for the job that you created at the start of the year then click on the ▶︎ button to run the job. Once you run the AutoCrat job, it will create the profile doc and add the information in columns M through P.

Finding the Google Drive Folder ID
To find the Google Drive folder ID, navigate to the folder in Google Drive and look at the URL in your browser's address bar. The folder ID is the string of letters and numbers that appears after /folders/ in the URL. For example, if the URL is https://drive.google.com/drive/folders/1dyUEebJaFnWa3Z4n0BFMVAXQ7mfUH11g, the folder ID is 1dyUEebJaFnWa3Z4n0BFMVAXQ7mfUH11g.
Google Apps Script
